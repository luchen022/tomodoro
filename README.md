-   简洁的用户界面

-   白噪音
-   主题切换
-   离线工作

## 任务和统计

![统计](https://lazy-guy.github.io/tomodoro/statistics.png)

番茄钟允许创建"任务"。专注时段花费的时间可以分配给任务以生成统计数据。
所有数据都保存在您的浏览器中。建议定期进行备份。备份和恢复选项可在番茄钟的设置中找到。

## 关于始终置顶/画中画模式

![画中画演示](https://lazy-guy.github.io/tomodoro/pip.png)

**未在 Safari 上测试，但可能可以使用**

画中画模式通过在画布上绘制、捕获其流并将其用作视频元素的源来工作。然后它请求画中画。

在桌面 Firefox 上，画中画按钮只会使视频可见，但不会激活画中画。要切换到画中画模式，请右键单击视频并选择"在画中画中观看"。

在 Android 浏览器上，例如不支持画中画 API 的 Chrome for Android，为了切换到画中画模式，请将视频全屏显示，然后转到您的设备主屏幕而不退出全屏。如果您的浏览器受支持，则将激活画中画模式。
